# 推荐软件

# Visual Studio Code（VS code）

地表最强代码编辑器！！！！

网址： https://code.visualstudio.com/ (opens new window)

# Typora

Markdowm编辑器

网址： https://www.typora.io/ (opens new window)

# Bandzip

开源压缩软件

网址：http://www.bandisoft.com/ (opens new window)

# Notepad++

文本编辑器

URL： https://notepad-plus-plus.org/ (opens new window)

# Cmder

命令行工具（如果你觉着CMD不好看，可以一试）

网址： http://cmder.net/ (opens new window)

配置：

1、添加环境变量： Path -> C:\cmder

2、配置右键快捷启动：cmd管理员身份 -> Cmder.exe /REGISTER ALL

个人百度云链接：https://pan.baidu.com/s/1OYd3A-8n3DktMoxzzUcLKQ (opens new window) 提取码：chjh

# Let's Encrypt

可以申请永久免费的SSL证书

网址： https://letsencrypt.org/ (opens new window)

# WIRESHARK

抓包工具

网址： https://www.wireshark.org/ (opens new window)

# Fiddler

HTTP调试抓包工具

网址： https://www.telerik.com/fiddler (opens new window)

# Postman

接口调试神器，后端大佬们接口还没写好的时候，这个工具用来给前端小伙伴mock一下api，还是蛮不错的哦~

网址：[ https://www.postman.com/ ] (opens new window)

# Lighthouse

自动化页面性能分析工具，会给你分析出当前页面哪些是需要优化的地方，并且提供优化方案~ 很实用

网址： https://developers.google.com/web/tools/lighthouse?hl=zh-cn (opens new window)

# Studio 3T（MongoDB）

Studio 3T是适用于Windows，Mac和Linux的MongoDB的专业GUI，IDE和客户端。

网址：https://studio3t.com/ (opens new window)

# Terminus

超级棒的终端！有功能！有颜值！

网址：https://github.com/Eugeny/terminus (opens new window)

# ScreenToGif

功能强大，体积小巧，使用简便的gif制作软件。

网址：https://www.screentogif.com/?l=zh_cn (opens new window)
