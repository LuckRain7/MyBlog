# 什么是事件？什么是事件绑定？

事件：

浏览器赋予元素天生默认的一些行为，不论是否绑定相关的方法，只要行为操作进行了，那么一定会触发相关的事件行为

事件绑定：

给元素的某一个事件行为绑定方法，目的是行为触发会可以做点自己想做的事情
