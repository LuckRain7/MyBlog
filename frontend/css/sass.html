      # sass

      世界上最成熟、最稳定、最强大的专业级 CSS 扩展语言！

      官网地址：https://sass-lang.com (opens new window)

      # 安装

      yarn add sass
# or
npm install -g sass

      # 使用

      # 定义变量 Variables

      $highlight-color: #F90;

      # 支持嵌套 Nested Rules

      #header {
  color: black;
  .navigation {
    font-size: 12px;
  }
  .logo {
    width: 300px;
  }
}

      # 循环语法

      @for $i from 1 to 20 {
    .p-#{$i} {
        padding: 5px * $i;
    }
}

      # Node Sass 弃用，以 Dart Sass 代替
